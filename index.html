표시 설정
남은 시간 표시?현재 교시의 남은 시간을 시계 아래에 표시합니다.
수업 시작 알림음?수업 시작 시 알림 차임벨을 재생합니다.
콜론 깜빡임?시계의 콜론(:)이 1초 간격으로 깜빡입니다.
초 시계 표시?시계에 초 단위까지 표시합니다.
시간표 표시?화면에 오늘의 시간표를 표시합니다.
쉬는 시간 음성 안내?쉬는 시간에 남은 시간을 음성으로 안내합니다. 아래에서 각 알림을 개별적으로 켜거나 끌 수 있어요.
알림장 여러 페이지 사용?알림장을 여러 페이지로 나눠서 쓸 수 있어요. 예: "아침 활동", "수업 시간", "알림장"을 각각 다른 페이지에 미리 써두면 지웠다 쓸 필요가 없어요. 끄면 기본 알림장 1개만 사용합니다.
음성 미리듣기
요일별 교시 수?각 요일마다 몇 교시까지 수업이 있는지 설정합니다.
시간표 편집?각 교시의 시작·종료 시간을 직접 수정하거나 자동 생성할 수 있습니다.
시작 시간
학교 선택 (나이스 연동)?학교명을 검색해 자기 학교를 선택하면, 학사일정과 오늘의 급식이 자동으로 표시됩니다. 설정은 각자 브라우저에만 저장되어 다른 선생님 설정과 섞이지 않습니다.
학교 이름의 일부만 입력해도 돼요. 예: "창원초" → 창원으로 시작하는 학교 목록 표시
학사 일정 / 날짜별 시간표?특정 날짜에 별도 일정과 전용 시간표를 설정합니다. 해당 날짜에만 자동 적용됩니다. 나이스 학교를 선택해두면, 직접 입력하지 않은 날에도 학사일정이 왼쪽 패널 배너에 자동으로 표시됩니다.
현재 선택: 새 일정 작성 중
1. 날짜와 일정 입력 2. 일정 저장 3. 아래 날짜 전용 시간표 수정. 배너는 해당 날짜 당일에만 표시됩니다.
날짜별 시간표
저장됨
과목 편집?시간표에 사용할 과목명과 색상을 추가·수정합니다.
학교종이 연동?학교종이 사이트 URL을 입력하면, 알림장 버튼 클릭 시 바로 알림장 작성 페이지로 이동합니다. 학교종이 로그인 후 주소창의 URL을 복사하여 붙여넣으세요.

자동 붙여넣기 북마클릿
아래 버튼을 브라우저 북마크바로 드래그하세요. 학교종이에서 클릭하면 알림장 내용이 자동 입력됩니다.
📋 알림장 붙여넣기
1. 위 버튼을 북마크바로 드래그
2. 알림장에서 "학교종이" 버튼 클릭 (내용 복사 + 페이지 이동)
3. 학교종이 페이지에서 북마크바의 "알림장 붙여넣기" 클릭
데이터 관리?설정과 시간표 데이터를 파일로 내보내거나, 백업 파일을 불러올 수 있습니다.
